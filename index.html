什么是 Token？

Token 是模型处理文本的基本单位。输入越长、输出越长，Token 消耗越多。

一个 Token ≈ 4 个字符。一个中文字 ≈ 1.5 Token。

五个节省技巧

  1. 一次说明完整需求。
  2. 只修改局部。
  3. 限制输出长度。
  4. 复用 Prompt 模板。
  5. 使用系统角色定位。

Prompt 模板

你是一位专业程序员。
直接给代码。
只修改需要修改的部分。
控制输出长度。

案例

将"重新生成整个网站"改为"只修改导航栏"，可减少 Token 消耗。
